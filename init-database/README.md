# INIT-DATABASE
## About
INIT-DATABASE is a script that allows you to manage the VAIN database.

Presents you with several options: 
- Do everything
- Drop database
- Create database
- Recreate database
- Read file
- Quit

```
Do everything: 
```


Requirements: 
---
- Node.js (v16.14.2)
- PostgreSQL 14
- **Makes sure a database called vain exists**

#### How to run:
1. run `npm install`
2. run `npm start`
